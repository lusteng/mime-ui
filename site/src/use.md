# 安装使用

`memui` 基于React定制的UI组件库,更多定制组件等待中...

## 安装

### 使用 npm 安装

```bash
$ npm install memui --save
```


## 使用示例

```js
import { Button } from 'memui'; 

ReactDOM.render(<Button />, mountNode);
```

引入样式：

```css
import 'memui/dist/index.css';  
```
